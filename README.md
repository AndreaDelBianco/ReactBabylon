# ReactBabylon

## Comandi

```
cd vite-project
npm install
npm run dev
```

## Configurazione tazza

Cose configurabili:

- colore (esterno, interno e della maniglia)
- materiale
- immagine o logo (multiple posizioni)
- grandezza
- testo
- forma
- accessori aggiuntivi (coperchio, cucchiaio o coaster)
